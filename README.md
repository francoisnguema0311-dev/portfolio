# 🌌 3D Solar System Portfolio

A high-fidelity, interactive 3D solar system built with **Three.js**, **GSAP**, and **Tailwind CSS**. Navigate through the cosmos to explore my skills, projects, and professional experience.

## 🚀 Features
- **Realistic 3D Experience**: High-resolution planetary textures and realistic orbital mechanics.
- **Interactive Navigation**: Click on any planet to dive deep into a specific portfolio section.
- **Simulation Control**: Adjust the speed of the solar system in real-time.
- **Smooth Transitions**: Cinematic camera movements powered by GSAP.
- **Glassmorphism UI**: A modern, clean overlay for portfolio content.

## 🛠️ Tech Stack
- **Core**: [Three.js](https://threejs.org/)
- **Animations**: [GSAP](https://greensock.com/gsap/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool**: [Vite](https://vitejs.dev/)

## 📦 Local Installation

1. Clone the repository:
   \`\`\`bash
   git clone https://github.com/your-username/solar-portfolio.git
   cd solar-portfolio
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   npm install
   \`\`\`

3. Run the development server:
   \`\`\`bash
   npm run dev
   \`\`\`

## 🌐 Deployment to GitHub Pages

This project is pre-configured with a GitHub Action to deploy automatically to GitHub Pages.

1. Push your code to the `main` branch of your GitHub repository.
2. Go to **Settings** $\rightarrow$ **Pages**.
3. Under **Build and deployment** $\rightarrow$ **Source**, select **GitHub Actions**.
4. The project will automatically build and deploy from the `dist` folder.
